# Bash 小抄

## 終端機快捷鍵
* `ctrl` + `L` = 清理終端機畫面
* `ctrl` + `D` = 離開終端機
* `shift` + `page up` = 向上移動終端機畫面
* `shift` + `page down` = 向下移動終端機畫面
* `ctrl` + `A` = 終端機游標移至該行的起始點
* `ctrl` + `E` = 終端機游標移至該行的結束點
* `ctrl` + `U` = 刪除在終端機游標左邊的全部文字
* `ctrl` + `K` = 刪除在終端機游標右邊的全部文字
* `ctrl` + `W` = 刪除在終端機游標左邊的部分文字，以空白為區隔
* `ctrl` + `Y` = 黏貼
* `tab` = 自動完成檔案或命令
* `ctrl` + `R` = 反向搜尋歷史紀錄
* `!!` = 再次執行最後一次所執行的命令
* `ctrl` + `Z` = 停止當前命令所執行的動作
